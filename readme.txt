SWT SEO Helper auto generating keywords, Adds Meta keywords and Description, facebook twitter google plus meta tags to each and every page, post, archive.
== Description ==

The first complete free all in one SEO solution for WordPress websites, including home page, archive pages, single post page etc. you can also set custom keywords, social media links for your hole website. This plugin helps you in SEO by adding Meta keywords, Description, facebook twitter google plus meta tags to each and every page, post, archive. (category, tag, date, day and year). This plugin also allows you to specify common keywords to be included on every web page. Plugin from one of the best coder <a href="http://www.phpkida.com/mukesh-jakhar-wordpress-php-web-developer-and-designer/" target="_blank">Mukesh Jakhar</a>. For any support just leave your question at our <a href="http://www.phpkida.com/swt-seo-helper/" target="_blank">SWT SEO HELPER PLUGIN</a> page.
Uses Title, Description, Categories, tags, Compulsory keywords (provided by you in setting) for keywords generation.

== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to page (wp-seo) and manage your website seo title, description, keywords and social meadia link.
